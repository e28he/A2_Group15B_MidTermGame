# A2 Midterm Game - Octopus Barista

Single-screen p5.js game prototype implementing the assignment mechanics:
- Four tentacle selection (`W/A/S/D`)
- Manual arm reach with mouse to stations
- Step-based execution per ingredient
- Micro-challenges (timing / hold / rhythm / tap)
- Tangled state + active untangle recovery (`SPACE` mash)
- Visible order queue + tangle meter + stress + score

## Run

Open `index.html` in a browser.

## Controls

- `W` = top-right arm
- `A` = top-left arm
- `S` = bottom-left arm
- `D` = bottom-right arm
- Move mouse to reach station
- `SPACE` = challenge input + untangle mash
- Hold `R` = reduce tangle meter
- `X` = clear selected arm / reset selection

## Objective

Complete orders before time runs out and keep coordination stable.
